Installation

Download the latest TouchBinder.zip file from the GitHub. After extracting the files, place your settings.txt file (made to the right) in the same folder as your executable. After running TouchBinder.exe, you will see a touch controller icon in the system tray. You can right click on the icon to exit the application or reload the settings file.

Basic Use

The first column lets you select what type of input you want. Pressed means send the keys once when the button is first pressed, released means once when the button is first released, and down means any time the button is pressed. The next column lets you select the specific input you wish to use. The "+" and "-" buttons can be used to add more required inputs in order to create more complex actions. The text input box contains the keyboard events to be executed, as described in the formatting guide below.

Formatting Guide

Key formating is identical to ahk notation. See the official docs for a full rundown. Here's a quick rundown:

  • Use ^,+,!, and # for ctrl, shift, alt, and win respectively
  • Put special items in curly brackets

Examples:

  • ^+a: Sends ctrl shift a
  • !{Tab}: Sends alt tab
  • {Media_Play_Pause}: Sends play/pause
